## About The Project

This project is aimed to built a dynamic webpage using “vanilla” JavaScript of an existing API. It showcases three main sections; work, bank and laptops.

The bank is an area where user can store funds and make bank loans.
The work is an area where user increases earnings and deposit cash into the bank balance.
The laptop is an area where user can select and display information about the merchandise.

Bank:

1. The bank shows a bank balance in dollar currency. This is the amount available to let user buy a laptop.
2. The bank shows the outstanding loan value. This reduces once the loan is paid back.
3. The bank shows the option to get a loan.
   3.1 User enters the desires amount to borrow.
   3.2 User cannot get a loan moer than double the bank balance.
   3.3 User cannot get more than one bank loan before paying the last loan.

Work:

1. The work shows a salary balance in dollar currency. This is the amount available deposit or repay the loan.
2. The work shows the option to repay the loan.
3. The work shows the option to deposit the salary balance into the bank balance.
   3.1 An interest rate of 10% is cut off from the salary.
   3.2 The balance after the 10% deduction is transferred to the bank balance.

Laptops:

1. Information is retrieved from an API.
2. Its information is displayed on the interface of the user.
3. User has the option to buy a laptop once there is sufficient funds on the bank balance.

Created by Ahmad Khodabaks.

1. Download .NET 6.0 Framework [https://dotnet.microsoft.com/en-us/download/dotnet](https://dotnet.microsoft.com/en-us/download/dotnet)
2. Clone the repo
   ```sh
   git clone https://gitlab.com/PauliusAle/MovieCharactersAPI.git
   ```

3. Run the program by clicking on F5 or on start debugging.
